# Casa 3D — Home Assistant add-on

An interactive low-poly 3D digital twin of your home, served as a sidebar panel
inside Home Assistant. Lights, covers, climate and sensors reflect live state;
tap a device in 3D to control the real thing.

## Install
1. In Home Assistant: **Settings → Add-ons → Add-on store**.
2. Top-right menu (⋮) → **Repositories** → paste this repo's GitHub URL → **Add**.
3. Find **Casa 3D** in the store, open it, **Install**, then **Start**.
4. Enable **Show in sidebar**. Open it from the sidebar.

> Add-ons require **Home Assistant OS** or **Supervised**. They do not run on
> Home Assistant Container or Core.

## First run
With no entities mapped yet, Casa 3D starts in **demo mode** so you can see it
working. To bind it to your real home, see `casa3d/DOCS.md`.
